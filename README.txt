*Installation*
Download the entire folder that contains this README file. This program suite requires Python 3 and a few extra libraries. After installing Python 3, install the required libraries from the command line by running the commands below:

pip install wxpython
pip install matplotlib
pip install scipy

*Usage*
In a terminal (PowerShell is recommended on Windows and comes preinstalled), navigate to the folder containing this README file and run the following command:

python ./run.py
