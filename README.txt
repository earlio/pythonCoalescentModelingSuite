# EZ 2018. This program suite was updated to work with Python 3 by Earl Zedd (earl.io) and has been tested on Windows 10. See the other files in this directory for authorship and copyright information.

*Installation*
Download the entire folder that contains this README file. This program suite requires Python 3 and a few extra libraries. After installing Python 3, install the required libraries from the command line by running the commands below:

pip install wxpython
pip install matplotlib
pip install scipy

*Usage*
In a terminal, navigate to the folder containing this README file and run the following command:

python ./run.py
